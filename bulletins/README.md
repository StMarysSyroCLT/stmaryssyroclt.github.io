# Bulletins

Upload bulletin PDFs to this folder.

After adding a PDF, add a matching link in `pages/about/bulletins.html`. Use URL-encoded spaces in links, for example:

```html
<a href="../../bulletins/Weekly%20Schedule%20%2806%3A01%3A2026%20-%2006%3A07%3A2026%29.pdf" target="_blank" rel="noopener noreferrer">Open PDF</a>
```
